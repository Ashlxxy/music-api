# API Musik - Tubes Kelompok 2 WebProPBO

Selamat datang di proyek API Musik yang dikembangkan oleh Kelompok 2 pada mata kuliah Web Programming dan Pemrograman Berorientasi Objek (PBO). Proyek ini bertujuan untuk menyediakan API yang memungkinkan pengguna untuk mengakses informasi terkait musik.

## Fitur Utama

- **Akses Data Musik**: Dapatkan informasi tentang lagu, artis, album, dan genre.
- **Pencarian**: Fitur pencarian untuk menemukan lagu atau artis berdasarkan nama.
- **Pluralisasi Kata**: Mendukung pluralisasi kata dalam respons API.
- **Data Real-Time**: Menggunakan teknologi modern untuk memberikan data yang up-to-date.

## Tumpukan Teknologi

Proyek ini menggunakan teknologi berikut:

- **HTML**: 51.5%
- **JavaScript**: 40.2%
- **CSS**: 8.3%
- **Python** (untuk backend, sesuai dengan contoh di README)

## Penggunaan API

Berikut adalah beberapa contoh endpoint yang tersedia:

### 1. Get All Songs
- **Endpoint**: `/api/songs`
- **Method**: `GET`

### 2. Search Songs
- **Endpoint**: `/api/songs/search?query={query}`
- **Method**: `GET`

### 3. Get Artist Info
- **Endpoint**: `/api/artists/{id}`
- **Method**: `GET`
